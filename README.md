# Cabinet IA — Repo central

Ce dépôt contient la plateforme du cabinet IA :

- workflows n8n
- schémas et migrations Supabase
- documentation fonctionnelle et méthodologique
- (plus tard) front React / cockpit

## Règles générales

- Pas de secrets dans le repo
- Pas de données clients non anonymisées
- Tout nouveau pipeline / idée / doc passe ici

Ce repo sert de socle pour rendre le cabinet réplicable, maintenable et documenté.

## Comment contribuer (moi ou quelqu’un d’autre)

1. Toute nouvelle idée / pipeline / doc commence ici.
2. Créer un dossier ou fichier au bon endroit.
3. Documenter le minimum pour qu’un autre puisse comprendre.
4. Ne jamais mettre :
   - secrets
   - tokens
   - données clients non anonymisées

## Scope du repo

Ce repo représente la plateforme du cabinet IA :
- pensée pour être réutilisable
- documentée
- sécurisée
- évolutive
