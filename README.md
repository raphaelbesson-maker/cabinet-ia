# Cabinet IA — Repo central

Ce dépôt contient la plateforme du cabinet IA :

- workflows n8n
- schémas et migrations Supabase
- documentation fonctionnelle et méthodologique
- (plus tard) front React / cockpit

## Règles générales

- Pas de secrets dans le repo
- Pas de données clients non anonymisées
- Tout nouveau pipeline / idée / doc passe ici

Ce repo sert de socle pour rendre le cabinet réplicable, maintenable et documenté.
